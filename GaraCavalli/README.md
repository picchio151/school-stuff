#  Gara dei Cavalli

**Versione:** 1.4
**Autore:** Picioccolo Matteo
**Titolo:** Simulazione di gara tra cavalli in Java come thread

##  Descrizione
Questo programma simula una gara tra più cavalli utilizzando un approccio orientato agli oggetti in Java.  
Ogni cavallo percorre una certa distanza a ogni turno e può azzoppare durante la corsa, smettendo di avanzare.  
La gara prosegue finché uno dei cavalli raggiunge il traguardo.  

L’intero svolgimento della gara (stato dei cavalli, vincitore, eventuali azzoppature) viene salvato anche in un file di testo chiamato **`risultato_gara.txt`**.

##  Funzionalità principali
- Gestione dei cavalli tramite `ArrayList`.
- Simulazione del movimento a intervalli regolari.
- Possibilità casuale che un cavallo si azzoppi (10% di probabilità).
- Scrittura dei risultati su file testuale.
